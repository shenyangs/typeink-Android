# 设置中心与多厂商支持架构设计

## 概述

实现对标 参考键盘 的设置中心，支持 10+ ASR/LLM 厂商的灵活配置。

## 架构设计

### 1. 核心模块

```
settings/
├── SettingsActivity.kt           # 设置中心主页面
├── SettingsViewModel.kt          # 设置状态管理
├── data/
│   ├── SettingsRepository.kt     # 设置数据仓库
│   └── PreferenceManager.kt      # SharedPreferences 封装
├── model/
│   ├── ProviderConfig.kt         # 厂商配置数据类
│   ├── AsrProvider.kt            # ASR 厂商枚举
│   └── LlmProvider.kt            # LLM 厂商枚举
└── ui/
    ├── ProviderListFragment.kt   # 厂商列表页面
    ├── ProviderEditFragment.kt   # 厂商编辑页面
    └── SettingsAdapter.kt        # 设置列表适配器
```

### 3. 配置数据结构

```kotlin
data class ProviderConfig(
    val id: String,                    // 厂商唯一ID
    val name: String,                  // 显示名称
    val type: ProviderType,            // ASR / LLM
    val isEnabled: Boolean,            // 是否启用
    val apiKey: String,                // API Key
    val apiSecret: String?,            // API Secret (部分厂商需要)
    val baseUrl: String?,              // 自定义 Base URL
    val model: String?,                // 模型名称
    val params: Map<String, String>,   // 额外参数
    val priority: Int                  // 优先级（用于 fallback）
)

enum class ProviderType {
    ASR, LLM
}
```
