# 构建与输出规范

## APK 输出路径（固定）

**唯一正确路径：**
```
/Users/sam/Desktop/typeink-codex/outputs/
```

**禁止路径：**
- ❌ `android/outputs/` - 临时构建目录，不要给用户
- ❌ `android/app/build/outputs/apk/debug/` - Gradle 默认输出

## 构建脚本规范

### 手动构建流程
```bash
cd /Users/sam/Desktop/typeink-codex/android
./gradlew :app:assembleDebug
cp app/build/outputs/apk/debug/app-debug.apk ../outputs/typeink-v${VERSION}-build${N}.apk
```

### 版本命名规则
- 格式：`typeink-v{VERSION}-build{N}.apk`
- 示例：`typeink-v0.3.11-build4.apk`
- build 号自动递增，不覆盖旧版本
